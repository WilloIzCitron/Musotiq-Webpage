Music Commands


1.Play - {Music}or{youtube video link} - Plays the music you have chosen!

2.Pause - it will pause the current music that is being Played.

3.Resume - Resumes the music after running the pause command

4.Skip - Skips to the next song. (NOTE: We added a nice feature that makes the bot autoplay by it self)

5.Loop - Makes the song that is being played never stopped unless you type run the command again

6.Stop - Stops the current Music being played

7.queue - shows the Playlist of some music you added to play after each song

8.volume - Makes you choose how high and low the volume of the bot to be

9.filters - Shows bunch of filters you can use while listening to music, for example (nightcore, bassboost, etc.)

10.autoplay - since we added this feature to random music once there is nothing left from the , you can type this command to disable the autoplay by typing this command



Other Commands


1.Add - Shows the Invite link to add the bot to different servers

2.stats - Shows how many servers is the bot

3.credits - Shows the creators of this bot

4.stats - shows how many servers this bot in and the amount of people using it

5.suggestion - use this command to suggest something and send it to our support server (you can find our support server by typing M!support)

6.supportserver - this will send the our support server to join for announcments, help, and so on

7.vote - this will send a link to vote for the bot and increase its popularity
